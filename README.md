适用：手机端vue框架，自适应各种手机屏幕；

代码说明：copy源码后，只需在根目录下cmd执行命令 npm install 安装依赖包即可跑起来;

功能说明：
    -标准的文件构建目录
    -头部和脚部共用
    -登录控制
    -非登录端与登录端view主组件区分
    -axios封装成ajax的数据请求方式，支持跨域
    -语言国际化
    -登录密码加密
    -less工具快捷写css
    -阿里巴巴的iconfont支持
    -vue-animate动画效果支持
    -mock模拟数据
    -vux作为ui组件引入
    -lib-flexible和px2rem-loader淘宝插件实现移动端伸缩自适应
